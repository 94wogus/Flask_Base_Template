# Flask_Base_Template
python3를 사용합니다.
## Project setup
```
python3 -m venv venv
source venv/bin/activate (linux & mac)
call venv/Script/activate (window)

pip install -U pip
pip install -U setuptools
pip install -r requirements.txt
```

### Compiles and hot-reloads for development
```
flask run
```
